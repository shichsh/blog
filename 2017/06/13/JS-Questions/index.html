Basic JavaScript questions

  1. What is the difference between == and ===?
    For Primitive type, == checks their values, === will check their types first and then check their values.
    For objects, both == and === compare their references

  2. What is closure? What is the drawback(disadvantage) of it? memory leak
    JS closure is a function that is defined in another function. The inner function can access all the local variables of the outer function. If the inner function is returned, then the local variable of the outer function is still alive after the outer function is executed.
    Disadvantage: Closure might result memory leak

  3. Difference between promise and deferred
    A promise is an object that may produce a single value some time in the future.It has then() function to add callback function.
    Defer is a Promise and defer hold parameter. Only JQuery and Angular has defer(it is Promise too). JS only has Promise.

    Each object has a prototype and inherits all the properties from its prototype. The object prototype is also an object and thus has its own prototype. Thus we get a prototype chain until Object.prototype, whose prototype is null

  6. Explain event propagation
    3 phases of DOM events: Capturing, Targeting, Bubbling
    • Capturing Phase – When an event occurs in the DOM, notification of the event is passed starting at the top of the DOM tree and passing down through all parent element nodes all the way to the target node where the event occurred.
    • Target Phase – After the capturing phase occurs, the Target phase occurs. The target phase only includes a notification of Node where the event took place.
    • Bubbling Phase – This is the final phase to occur, although many people think this is the first phase. In the bubbling phase a notice is passed from the target Node up through all of the parent Nodes all the way back to the top root of the DOM.
